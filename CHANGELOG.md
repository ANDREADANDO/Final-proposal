# Changelog for the Project Proposal

> NOTE: This file lists all the changes we made to our Savings Tracker.

## Version 1.0.0 — August 15, 2025
Initial release of Savings Tracker.

- Allows input for monthly savings over 12 months  
- Displays total savings at the end of the year  


## Version 1.1.0 — December 4, 2025
Added BONUS feature.

- Users can now choose if they received a bonus  
- Bonus amount is added to total yearly savings  


## Version 1.2.0 — February 14, 2026
Added Emergency Withdrawal Feature.

- Users can now withdraw money for emergencies  
- Prevents negative savings by setting total to $0 if withdrawal exceeds balance  
- Improved conditional logic with nested decision handling  
- Updated documentation to reflect new feature  
 

## Version 1.3.0 — April 2026
Added JSON Data Storage Feature.

- Program now saves all user inputs and results into a JSON file (`savings_record.json`)  
- Allows saving of multiple records (history tracking)  
- Improves data persistence and record management  
- Enables future review of past savings entries
